# TaskFlow 库

这是一个临时的最小化 TaskFlow 包装。

## 获取完整的 TaskFlow 库

为了获得完整的 TaskFlow 功能，请运行：

```bash
git submodule update --init --recursive
```

或者手动下载 TaskFlow 到当前目录：

```bash
git clone --depth 1 --branch v3.8.0 https://github.com/taskflow/taskflow.git .
```

## 跨平台支持

TaskFlow 支持 Windows、macOS 和 Linux，通过 git submodule 管理可以确保在所有平台上使用相同的版本。
